Week 4: Javascript

What is Javascript?

Javascript is an object-based scripting language.
It can manipulate objects associated with the Web page document, such as, the window, the document and it's elements.
It was originally developed by Netscape, and was named LiveScript.
Following the Netscape & Sun Microsystems Collab, LiveScript was renamed to Javascript.

Coding Javascript

Javascript statements can be coded usig two different techniques.
- Embedded script
- External script
Embedded
To code Javascript embeddedly we can use the <script>. The script element can be placed either at the head or body of webpage.
Example 
<script>
alert("Welcome to MySite");
</script>
External
To code Javascript as an external script, we still use the <script>. However we must declare its pseudo-element src to point towards to the external file.
Example 
<script src="myJavaScript.js"></script>

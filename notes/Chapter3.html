Week 3 : Web Styling

Components in Webpage

HTML: HyperText MarkUp Language

Gives stucture and meaning to content and links to other web pages

CSS: Cascading Style Sheets

Defines the presentation (look and feel) of your site

JS: Javascript

Adds interactivity to your site

Cascading Style Sheet (CSS)

CSS is a language to design and create great-looking website.
With CSS you can define how HTML elements are displayed using styles.

Styles

A style is a rule that describes how to format a part of an HTML document.
A stylesheet is a set of style rules.

CSS Types

Inline
Example: <p style = "font-size: 14px;"></p>
Appears within a tag and only applies to that tag.
PROS:
Highly specific to the element on which it is defined
Don't need selector
Can used to override styles defined elsewhere
CONS:
Hard to maintain: Bad for reusability because need to reapply the style to every single tag
HTML/CSS coupled: Violates the primary goal of separation
Embedded (Internal / Block)
Example 
<style>
  p {
    font-size: 14px;
  }
</style>
Defined within the head of a page and applies to that page only.
PROS:
Affects all matched elements
Used to have a stand-alone webpage that contains everything needed to render
CONS:
HTML and CSS coupled: Still does not provide file seperation. Only reuse within the files.
Need to use SELECTORS
External
Example 
p {
  font-size: 14px;
}
(Placed in seperate file)
Defined in an external file and applied to all HTML document that are linked.
PROS:
Easy to maintain: Write once for whole site
HTML and CSS decoupled: Allowing for themes

Selectors

Selectors connect style rules to your HTML.
Example 
selector {
  property: value;
  property: value;
  ....
}
Selectors can be used to select based on:
Element type [div]
Id [#name]
Classes [.name]

Element-type selectors

An element type selector is based on the name of the tag.
Styles will be applied to every type-of selected element.
Example: [div], [buttons]

Id selectors

An id selector is based on the id of the element and is prefixed with the # symbol.
Id's are unique across a HTML document. No two elements can have the same id.

Class selectors

A class selector is a style with a class name of your choice and is prefixed with the period (.) symbol.
Class styles promotes reuse as they can used on any element as needed.

Selector chain

Style Inheritance

There exists many ways to chain selectors in CSS.
Example 
li a {
  text-decoration : none;
}
The above selector selects all <a> that are descendants of a <li>.
However, if we only want to remove the underline of <a> that are only direct childrens of a <li>, we should use the > symbol 
li >a {
  text-decoration : none;
}

Margin is the space outside the border of the element and the next element.
Padding is the space inside the border, between the border and the content.
Border is the box enclosing the content.

<div>

The <div> defines a division in a HTML document.
It can be used to group block elements annd format them with CSS.
<div> is the all purpose tag used for creating page layouts.

<div> positioning

The static position: The element is displayed where it would normally appear in the HTML flow.
The relative position: The element can be offset from where it normally appears in the HTML flow.
The absolute position: The element is removed from the HTML flow and is positioned within the first non-static element.
The fixed position: The element is calculated similarly to absolutely positioning, but is always in relation to the browser window.

Float property

The float property positions an element to the left or right side of a container
Useful in placing elements side by side to create columns.
Can be ignored using the clear: property.

Boxing Property (box-sizing)

Box sizing has the following values:
content-box: Calculates the width based on content width only. Default setting.
border-box: Calculates the width based on the border, padding and content width.
padding-box: Calculates the width based on the padding and content width only.
